# 🚀 FUTURE_FS_01 | High-Fidelity SDE Portfolio

[![Live Site](https://img.shields.io/badge/Live-Portfolio-blue?style=for-the-badge&logo=vercel)](https://porfolioo-eta.vercel.app)
[![Digital Resume](https://img.shields.io/badge/Digital-Resume-purple?style=for-the-badge)](https://porfolioo-eta.vercel.app/resume)
[![Tech Stack](https://img.shields.io/badge/Next.js%2015-Black?style=for-the-badge&logo=next.js)](https://nextjs.org/)

A professional, high-impact Software Development portfolio designed for impact and performance. Built with **Next.js 15**, **Tailwind CSS v4**, and **Three.js**, this site showcases technical expertise in AI, Machine Learning, and Full Stack Development.

---

## ✨ Key Features

- **🛡️ High-Fidelity Design**: Modern glassmorphic UI with hardware-accelerated WebGL backgrounds (Three.js Shader Animations).
- **📄 Interactive Digital Resume**: A dedicated `/resume` route that serves a high-fidelity, printable CV, bypassing the need for manual PDF uploads.
- **⚡ Performance First**: Fully optimized using Next.js 15 App Router and Framer Motion for buttery-smooth 60fps micro-animations.
- **📱 Fully Responsive**: Tailored experience across mobile, tablet, and desktop with a focus on premium SDE aesthetics.
- **🛠️ Project Showcases**: Featured cards for **AstraSense** (Environmental AI), **Smart Civic Monitoring**, and **AgriSense AI**.
- **🏆 Hackathon Log**: A serialized log of national-level hackathon achievements (IIM Indore, AWS, VMC).

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Graphics**: [Three.js](https://threejs.org/) (WebGL Shaders)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)

## 🚀 Getting Started

Follow these steps to set up the project locally:

### 1. Clone the Repository
```bash
git clone https://github.com/pranavpatil179/FUTURE_FS_01.git
cd porfolioo
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 4. Build for Production
```bash
npm run build
```

---

## 🚢 Deployment

The project is pre-configured for **Vercel** with automatic CI/CD:

1. Connect your GitHub repository to Vercel.
2. The `next build` command is automatically detected.
3. Every push to `main` triggers a production deployment.
